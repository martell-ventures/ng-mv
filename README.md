### How to tag versions
Be sure package.json and bower.json have the correct (new) version.

To list (to find current version):
```
git tag -l
```

To tag (full tag):

```
git tag -a v0.0.66 -m "Send token from FB when logging in"
```

And to push:
```
git push origin --tags
```
